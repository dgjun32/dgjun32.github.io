# Learning to Generate Unit Test via Adversarial Reinforcement Learning

Project page for ICLR 2026 paper.

## Project Structure

```
UTRL/
├── index.html           # Main project page
├── assets/
│   ├── css/
│   │   └── main.css    # Styling
│   └── imgs/           # Place your images here
│       ├── overview.png    (recommended: ~800px wide)
│       ├── problem.png     (recommended: ~800px wide)
│       ├── method.png      (recommended: ~800px wide)
│       └── results.png     (recommended: ~800px wide)
└── README.md
```

## Setup

1. Add your images to `assets/imgs/` directory
2. Update the following placeholders in `index.html`:
   - Author names and links
   - Institution/affiliation
   - arXiv link (line 82)
   - GitHub repository link (line 88)
   - HuggingFace dataset link (line 94)
   - HuggingFace model link (line 100)
   - BibTeX citation (lines 527-532)

3. Customize the content:
   - Overview section: Describe your approach
   - Problem section: Explain the challenges
   - Method section: Detail your technical approach
   - Results section: Add your experimental results and examples

## Recommended Image Sizes

- **overview.png**: 800x400px or similar aspect ratio
- **problem.png**: 800x500px (for comparison charts/tables)
- **method.png**: 800x600px (for architecture diagrams)
- **results.png**: 800x400px (for results tables)

All images should be in PNG or JPG format with reasonable file sizes (< 500KB each).

## Deployment

This site is designed to be hosted on GitHub Pages. Simply push to your repository and enable GitHub Pages in the repository settings.
